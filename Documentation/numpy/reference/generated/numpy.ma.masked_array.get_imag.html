numpy.ma.masked_array.get_imag¶

masked_array.get_imag()[source]¶

Return the imaginary part of the masked array.

The returned array is a view on the imaginary part of the MaskedArray whose get_imag method is called.

Parameters:

None :
Returns:

result : MaskedArray

The imaginary part of the masked array.

See also

get_real, real, imag

Examples

>>> x = np.ma.array([1+1.j, -2j, 3.45+1.6j], mask=[False, True, False])
>>> x.get_imag()
masked_array(data = [1.0 -- 1.6],
             mask = [False  True False],
       fill_value = 1e+20)
